# RFC-0012.6 — Misaka Advanced Voice, Web Actions & Final Desktop Pack

**Status:** Implemented  
**Version:** 0.3.5 Genesis  
**Date:** 2026-07-02  
**Branch:** `feature/rfc-0012-6-advanced-voice-web-actions`

---

## Summary

Transform Misaka's command system from basic keyword matching to a full web action engine capable of understanding natural language commands for opening sites, searching YouTube/Google/GitHub/Reddit, and executing real actions on the PC.

## What Changed

### 1. Web Action Engine (`backend/app/web_actions/`)

New module that parses natural language into structured web actions:

- **parser.py** — Extracts entities: target_site, query, content_type, creator, topic, action_type
- **engine.py** — Processes messages and builds URL actions
- **url_templates.py** — URL templates for 10+ sites (YouTube, Google, GitHub, Reddit, Wikipedia, Twitch, Steam, CurseForge, Modrinth, NexusMods)
- **schemas.py** — WebAction and ParsedEntities dataclasses

### 2. Supported Commands

| Command | Action |
|---------|--------|
| "abra o youtube" | Opens YouTube |
| "abra o canal do alanzoka no YouTube" | YouTube channel search |
| "abra um vídeo do alanzoka de minecraft" | YouTube video search |
| "pesquise X no YouTube" | YouTube search |
| "pesquise X no Google" | Google search |
| "pesquise X no GitHub" | GitHub search |
| "pesquise X no Reddit" | Reddit search |
| "procure cobblemon no Modrinth" | Modrinth search |
| "procure mods no CurseForge" | CurseForge search |
| "abra o discord" | Opens Discord (Electron) |
| "abra o vs code" | Opens VS Code (Electron) |
| "abra o explorer" | Opens Explorer (Electron) |
| "limpe os alertas atuais" | Clears alerts |
| "ative o modo HUD" | Toggles HUD |
| "abra configurações" | Opens settings |

### 3. Command Router Priority

1. Web Action Engine runs first (highest priority)
2. Intent-based Command Router runs second
3. LLM fallback runs last

### 4. Client Action Metadata

Every command returns `client_action` in metadata:

```json
{
  "intent": "web_action",
  "command": "open_youtube_channel_search",
  "client_action": {
    "type": "open_url",
    "target": "desktop",
    "url": "https://www.youtube.com/results?search_query=alanzoka+canal"
  }
}
```

### 5. Dashboard Execution

`handleClientAction()` in dashboard/app.js executes:
- `open_url` → `misakaDesktop.openUrl()` or `window.open()`
- `open_app` → `misakaDesktop.openApp()` or warning
- `search_web` → `misakaDesktop.searchWeb()` or `window.open()`

### 6. Desktop IPC

New IPC handlers in desktop/main.js:
- `desktop:open-url` — Opens URL via shell.openExternal
- `desktop:open-app` — Opens app via allowlist
- `desktop:search-web` — Searches via provider-specific URL

Preload exposes: `openUrl`, `openApp`, `searchWeb`, `getSystemStatus`, `setHudMode`, `setAlwaysOnTop`
