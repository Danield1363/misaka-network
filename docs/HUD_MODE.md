# HUD Mode - Misaka Network

## Overview

Modo HUD (Heads-Up Display) torna a interface translúcida e flutuante, ideal para overlay.

## Funcionalidades

- Fundo translúcido
- Painéis de vidro (glassmorphism)
- Anéis ativos ao fundo
- Brilho suave
- Sem moldura de janela (desktop)

## Como ativar

1. Clique no ícone ◎ no header
2. O modo será salvo localmente
3. Recarregue a página para manter a preferência

## Visual

- Background muito sutil
- Painéis com transparência
- Efeitos de glow
- Bordas arredondadas
- Animações suaves

## Regras

- Não sacrifica legibilidade
- Pode ser alternado a qualquer momento
- Preferência persistida no localStorage
